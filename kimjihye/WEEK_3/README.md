
# 3장. 템플릿

>개방폐쇄원칙 OCP
> 1
> 코드에서 어떤 부분은 변경을 통해 그 기능이 다양해지고 확장하려는 성질이 있고,
> 어떤 부분은 고정되어 있고 변하지 않으려는 성질이 있음
> 2
> 변화의 특성이 다른 부분을 구분해주고,
> 각각 다른 목적과 다른 이유에 의해 다른 시점에 독립적으로 변경될 수 있는 효율적인 구조를 만들어주는 것
